# UML Diagrams

Các sơ đồ PlantUML cho project `TheAllNewBinance` được tách theo từng góc nhìn:

- `01-overview.puml`: tổng quan module `client`, `core`, `server`
- `02-domain.puml`: domain model chính
- `03-core-contracts.puml`: DTO, DAO interfaces, Service interfaces, protocol, utils
- `04-server.puml`: server-side class diagram
- `05-client.puml`: client-side class diagram
- `06-sequence-login.puml`: sequence diagram cho luồng login
- `07-sequence-place-bid.puml`: sequence diagram cho luồng place bid

File gốc tổng hợp vẫn nằm ở `TheAllNewBinance-overview.puml`.
